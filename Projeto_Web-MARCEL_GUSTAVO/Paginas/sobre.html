O Projeto DC Hero Hunter

📄 Descrição ⚙️ Tecnologias Utilizadas 👨‍💻 O Desenvolvedor

Desenvolvido por Marcel Gustavo, estudante de Engenharia de Software do IFPE - Campus Belo Jardim.

Trabalho apresentado como requisito para a disciplina de Desenvolvimento Web, sob orientação do professor Fábio Feliciano.

O objetivo deste sistema é permitir que os usuários explorem o vasto universo da DC Comics, pesquisem seus personagens favoritos e visualizem estatísticas detalhadas de combate e biografia, servindo como um portfólio prático de Desenvolvimento Web Front-End com 10 páginas estruturadas.
