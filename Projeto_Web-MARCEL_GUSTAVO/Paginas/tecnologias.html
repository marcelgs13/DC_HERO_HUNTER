O Projeto DC Hero Hunter

📄 Descrição ⚙️ Tecnologias Utilizadas 👨‍💻 O Desenvolvedor

HTML5 & CSS3

Estruturação semântica e estilização moderna com Flexbox e CSS Grid para total responsividade, utilizando paleta de cores escura inspirada na Batcaverna.

JavaScript

Lógica de programação pura, manipulação do DOM em tempo real, roteamento inteligente de páginas e algoritmos de filtro, pesquisa e ordenação (Ranking).

Fetch API & Async/Await

Consumo assíncrono da SuperHero API (Akabab) para buscar o JSON com os dados completos, imagens e atributos de combate dos personagens.

Local Storage

Uso do armazenamento do navegador para garantir que a sua lista de personagens Favoritos não desapareça mesmo que a página seja recarregada.
